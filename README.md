# README


This is an example using [cocoon](https://github.com/nathanvda/cocoon) to dynamic add with two nested options (Color and Dimension).


* Rails version : 5.0.2

* Ruby version : 2.3.0

* Gem : [cocoon](https://github.com/nathanvda/cocoon)

* How to run

  1. fork

  2. ```$ bundle install```

  3. ```$ rake db:migrate```

  4. ```$ rake db:seed```

  Have fun.
